# Responsive Login/Signup Form

A modern, mobile‑friendly login and signup form built with **HTML, CSS, and JavaScript**.  
Includes field validation, error handling, and a clean UI/UX design suitable for integration into websites and web apps.

---

## 🚀 Features
- Responsive design (works on desktop, tablet, and mobile)
- Input validation (email format, password length, required fields)
- Error messages with clear feedback
- Gradient styling and modern UI
- Easy to customize and integrate

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)
